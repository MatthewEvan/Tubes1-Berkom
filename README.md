# AutoGate Parking System - Tugas Besar 1 - WI1102 Berpikir Komputasional - K34

Sebuah program yang bertujuan untuk mensimulasikan cara kerja AutoGate Parking System yang digunakan oleh Mahasiswa ITB Jatinangor.
Tugas Besar ini berfokus pada dekomposisi, abstraksi, dan pengenalan pola dari AutoGate Parking System yang ada, untuk direkonstruksi
menjadi flowcharts dan program final dalam bentuk kode Python, yang berisi konsep dasar Sequence, Conditional, Looping, dan Array.
